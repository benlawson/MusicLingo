#Music

####by Ben Lawson
==================

The goal of this project is to create an easier way to manipultate music and query it. This language is built from using user-submitted data to allmusic.com. 
 
Expresions:
+ Length (song)
+ Lyrics (song)
+ Interval (song start end)
+ Style (song | artist) 
+ Artist (song) 
+ Element (word song) 
+ [ word ]

Statements:
+ Query (expression)
+ Print (expression)
+ Play  (expression)
